# 开发目标：工作小助手

## 技术规划

- 使用 vue3 作为前端组件框架，渐进式开发；
- 前期为熟悉 css，手写 css，原则是先功能后界面；
- css 掌握差不多后，开始使用成熟的界面框架；
- 后端前期使用 python 的 fastapi 异步框架，后期使用 go 语言重构；


## 功能构思

- 自动生成当天工作日志
- 未完成工作有清晰视图
- 正在进展工作 / 未完成 有进展进度和情况细节的一览图
